# 📧 Gmail to Excel Data Collection using Power Query
I am currently learning Excel, Power Query, Power BI, and Data Analytics. I enjoy building small automation projects that solve real-world problems and help improve productivity.<Br>
📌 Project Overview

This project shows how to collect email data from Gmail directly into Microsoft Excel using Power Query.

Instead of copying email data manually, Power Query automatically imports and refreshes the data, making the process faster and reducing manual work.
